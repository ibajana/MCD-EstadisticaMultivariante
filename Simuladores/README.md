# Simuladores Interactivos

Archivos HTML autocontenidos que permiten explorar visualmente los conceptos del curso. Se abren directamente en el navegador sin necesidad de instalar R ni ninguna dependencia adicional.

---

## Archivos

| Archivo | Tema | Descripción |
|---|---|---|
| `simulador_correlacion_pearson.html` | Correlación | Exploración visual del coeficiente de Pearson |
| `simulador_pca_step_by_step.html` | PCA | Análisis de Componentes Principales paso a paso |

---

## Descripción

### `simulador_correlacion_pearson.html`
Permite manipular puntos en un plano y observar en tiempo real cómo cambia el coeficiente de correlación de Pearson. Útil para construir la intuición sobre:
- Correlaciones positivas, negativas y nulas
- Efecto de valores atípicos sobre r
- Diferencia entre correlación lineal y asociación no lineal

### `simulador_pca_step_by_step.html`
Muestra el proceso completo de PCA de forma visual e interactiva:
1. Nube de puntos original
2. Centrado de los datos
3. Cálculo de la matriz de covarianza
4. Eigenvectores y eigenvalores
5. Proyección sobre las componentes principales

---

## Uso

Abrir directamente en cualquier navegador moderno (Chrome, Firefox, Safari):

```bash
open simulador_correlacion_pearson.html
open simulador_pca_step_by_step.html
```
