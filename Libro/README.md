## Descripción

### ISLR v2 — James, Witten, Hastie, Tibshirani
La bibliografía principal del curso. Cubre los métodos estadísticos vistos en clase con implementaciones en R:

| Capítulo | Tema |
|---|---|
| Cap. 2 | Aprendizaje estadístico — conceptos generales |
| Cap. 3 | Regresión lineal |
| Cap. 4 | Clasificación (logística, LDA, QDA, Naive Bayes) |
| Cap. 6 | Selección y regularización de modelos |
| Cap. 10 | Aprendizaje no supervisado (PCA, K-means, clustering jerárquico) |
| Cap. 12 | Análisis de componentes principales (profundidad) |

### ISLP — Versión Python
Equivalente del ISLR v2 con código en Python usando `scikit-learn`, `statsmodels`, `pandas` y `matplotlib`. Útil como referencia para los scripts `.py` del curso.
